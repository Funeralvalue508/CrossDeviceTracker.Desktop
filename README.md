## 📋 What is CrossDeviceTracker.Desktop?

CrossDeviceTracker.Desktop is a Windows desktop app that watches which programs you use. It keeps track of how long each app stays open and shares that data with a central system. This helps you see where your time goes on your computer. The app runs quietly in the background and collects screen time information to help you better manage your computer use.

It does the following:

- Tracks which window is active on your screen.
- Measures how long you spend on each application.
- Sends collected data to an online server for syncing.
- Runs on Windows computers with minimal setup.
- Uses a small, simple interface to show some basic stats.

## 🖥 System Requirements

CrossDeviceTracker.Desktop runs on most Windows computers. Make sure your PC meets these minimum requirements:

- Operating System: Windows 10 or later (64-bit).
- Processor: Intel or AMD, 1.5 GHz or faster.
- Memory: At least 2 GB of RAM.
- Disk Space: At least 100 MB free space.
- .NET Desktop Runtime: Version 6.0 or newer installed (you can get it from Microsoft's official site).
- Internet connection: Needed to sync your screen time log with the backend.

If you are unsure about the .NET Desktop Runtime, the app setup will notify you if it is missing.

## ⚙️ Background Process Information

CrossDeviceTracker.Desktop monitors which window is in the foreground on your screen. It tracks active window titles and timestamps to build a log of your computer activity. It uses Windows API calls to detect the foreground window.

The app sends this information securely to a server in regular intervals. This helps keep all your screen time records in one place if you use CrossDeviceTracker on multiple devices.
